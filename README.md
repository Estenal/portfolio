# Estenal's Portfolio

Chào mừng bạn đến với Estenal's Portfolio – một trải nghiệm giới thiệu bản thân độc đáo, được thiết kế theo phong cách gamification và tương tác 3D! Dự án này không chỉ là một trang web thông thường mà còn là một không gian sáng tạo, nơi tôi thể hiện các kỹ năng phát triển phần mềm và thiết kế của mình thông qua một giao diện đầy tính khám phá.

Được xây dựng với React, TypeScript và Three.js, portfolio này phá vỡ các khuôn mẫu truyền thống, mang đến một hành trình trực quan và hấp dẫn để khám phá các dự án, kỹ năng và hành trình phát triển của tôi.

## Features
- **Interactive 3D Environment**: Tích hợp Three.js để render các mô hình 3D, tạo ra một không gian ảo độc đáo để người dùng khám phá.
- **Gamified UI/UX**: Thiết kế giao diện người dùng theo phong cách trò chơi, kết hợp hiệu ứng âm thanh và chuyển động mượt mà (sử dụng Framer Motion) để tăng cường tính tương tác và sự thú vị.
- **Dynamic Project Showcase**: Trình bày các dự án của tôi trong một môi trường 3D, cho phép người dùng "bay" qua và tương tác với từng dự án để xem chi tiết.
- **Responsive Design**: Đảm bảo trải nghiệm mượt mà và nhất quán trên nhiều thiết bị khác nhau.
- **Modern Frontend Stack**: Sử dụng các công nghệ frontend hiện đại như React, TypeScript và TailwindCSS để xây dựng một ứng dụng mạnh mẽ và dễ bảo trì.
- **AI/LLMs Assisted Development**: Ứng dụng AI/LLMs trong việc hỗ trợ tối ưu hóa logic phức tạp và xử lý lỗi (debugging), nâng cao hiệu quả phát triển.
- **Performance Optimization**: Quản lý state và render hợp lý để đảm bảo hiệu suất mượt mà cho một trang web đa phương tiện (Rich Media).

## Technologies Used

### Frontend
- **React**: Thư viện JavaScript để xây dựng giao diện người dùng.
- **TypeScript**: Ngôn ngữ lập trình giúp tăng cường tính an toàn kiểu dữ liệu và khả năng mở rộng của dự án.
- **Three.js**: Thư viện JavaScript để tạo và hiển thị đồ họa 3D trên trình duyệt web.
- **Framer Motion**: Thư viện mạnh mẽ cho các hiệu ứng animation và tương tác mượt mà.
- **TailwindCSS**: Framework CSS tiện ích giúp xây dựng giao diện nhanh chóng và linh hoạt.

### Other Tools & Concepts
- **Git**: Quản lý mã nguồn và theo dõi phiên bản.
- **Visual Studio Code**: Môi trường phát triển tích hợp (IDE) chính.
- **Figma**: Công cụ thiết kế UI/UX để xem và trích xuất tài sản.

## Design Philosophy

Portfolio này được xây dựng dựa trên triết lý thiết kế tập trung vào trải nghiệm người dùng (UX) và tính tương tác. Tôi tin rằng người dùng không nên học cách sử dụng sản phẩm – sản phẩm phải tự giải thích chính nó. Mục tiêu là tạo ra một không gian khám phá, phản hồi trực quan và chuyển động có mục đích, nơi việc tương tác không chỉ hoàn thành tác vụ mà còn mang lại cảm giác thú vị.

## Development Journey

Dự án đã trải qua nhiều giai đoạn phát triển và thay đổi ý tưởng:
- **Khởi tạo (v0.1.1)**: Lên ý tưởng ban đầu về một portfolio cá nhân sử dụng Three.js và chuyển động theo scroll (Scroll Driven).
- **Phong cách DarkFantasy (v0.1.2)**: Thiết kế theo phong cách DarkFantasy với các mô hình kiếm, vương miện và ngai vàng.
- **Giai đoạn ngừng phát triển**: Một khoảng thời gian tạm dừng do vấn đề cá nhân và mất phương hướng.
- **Ý tưởng Rạp Xiếc (v0.2.1)**: Chuyển đổi sang ý tưởng rạp xiếc với intro dạng Idle Game và animation mở/đóng màn sân khấu.
- **GameChunky Style (v0.3.1)**: Thay đổi thiết kế sang phong cách GameChunky, tập trung vào tăng cường UX, thêm hiệu ứng, sức nặng và động năng của giao diện, cùng với sound effect và logic bay cho máy bay.
- **Deploy Phiên bản đầu tiên (v1.1.1)**: Triển khai phiên bản portfolio đầu tiên trên Vercel, hoàn thiện nội dung cho kỹ năng và dự án, cùng với âm thanh UI và hiệu ứng chuyến bay.

## Installation & Usage

Để chạy dự án này cục bộ:

1.  Clone repository:
    ```bash
    git clone https://github.com/your-username/estenal-portfolio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd estenal-portfolio
    ```
3.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
5.  Open your browser and visit `http://localhost:5173` (hoặc cổng được hiển thị trong terminal).
